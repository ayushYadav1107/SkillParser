# **App Name**: SkillParser

## Core Features:

- Resume Parsing: Parse resumes from various formats (PDF, DOCX) and extract relevant information.
- Job Description Analysis: Analyze the job description to identify key skills, experience, and requirements.
- Skill Extraction using NLP: Extract skills, experience, education, and certifications from resumes using Named Entity Recognition (NER).
- Resume Scoring: Calculate a matching score between the resume and the job description based on extracted information and text similarity analysis; an LLM tool can incorporate specific additional data, such as candidate availability.
- Ranking and Filtering: Rank resumes based on their matching scores and provide filtering options based on skills, experience, etc.
- ATS Integration (MVP): Provide basic integration capabilities with Applicant Tracking Systems (ATS) for seamless data transfer.

## Style Guidelines:

- Primary color: Dark teal (#008080) for a professional and trustworthy feel.
- Background color: Light gray (#F0F0F0) to provide a clean and neutral backdrop.
- Accent color: Soft green (#8FBC8F) for highlighting important information and CTAs.
- Body and headline font: 'Inter' for a modern and clean sans-serif feel.
- Use simple, professional icons to represent different skills, qualifications, and categories.
- Use a clean, well-organized layout with clear sections for resume details, job description, and matching scores.
- Subtle animations for loading states and transitions to enhance user experience.
